CourbesWin.exe : Exécutable pour Windows	(make)
CourbesLin.exe : Exécutable pour Linux		(make PLATFORM=linux)
Compilation nécessaire si modification du programme d'asservissement ou du PSoC
Les variables globales à afficher sont écrites dans Vars.c.
Pour voir les graphique, donner le fichier de sortie au Visionneur.
Le nom du fichier de sortie peut être donné en argument. Par défaut, il vaut "Output.bin".
